Ciclo de vida das Classes Components.

os principais métodos do ciclo de vida são:
-O método componentDidMount: É um método muito bom para realizar requisições http porque ele é executado apenas uma vez;
-O método componentDidUpdate; Acontece quando ocorre uma mudança no componente. Com isso podemos manipular a cada mudança.
-O método componentWillUnmount; Elimina o efeito colateral que o nosso componente esteja utilizando
